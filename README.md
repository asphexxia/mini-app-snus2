# Telegram Mini App B2P Склад

MVP мини-приложения для Telegram с каталогом по категориям и остатками из Excel.

## Что реализовано

- Категории: `снюс`, `жижа`, `одноразки`, `картриджи`
- Список товаров в формате: фото слева, название и остаток справа
- Увеличение фото товара по нажатию
- Кнопка `Предложения по поставке` с отправкой сообщения владельцу в Telegram
- Источник данных: `data/products.xlsx`
- Если в Excel у товара `stock = 0`, он автоматически скрывается
- Поддержка Telegram WebApp (`telegram-web-app.js`)

## Формат Excel

Файл: `data/products.xlsx`

Обязательные колонки в первой вкладке:

- `category` - одна из категорий (`снюс`, `жижа`, `одноразки`, `картриджи`)
- `name` - название товара
- `stock` - остаток (число)
- `image` - URL картинки

Пример строки:

| category | name | stock | image |
|---|---|---:|---|
| жижа | Lemon Mint 30ml | 5 | https://... |

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить сервер:

```bash
npm start
```

3. Открыть в браузере:

- `http://localhost:3000`

## Синхронизация с Excel

1. Открой `data/products.xlsx`
2. Измени остатки
3. Сохрани файл
4. Нажми кнопку `Обновить` в mini app

Сервер читает Excel на каждый запрос, поэтому актуальные остатки подтягиваются сразу после сохранения.

## Подключение в Telegram

Для реального запуска в Telegram Mini App нужен HTTPS домен (например, через VPS или Cloudflare Tunnel) и ссылка в боте на web app URL.

## Отправка предложений владельцу

Чтобы кнопка `Предложения по поставке` отправляла текст тебе в Telegram, нужно задать 2 переменные окружения перед запуском сервера.

PowerShell (в папке проекта):

```powershell
$env:TELEGRAM_BOT_TOKEN="ТВОЙ_ТОКЕН_БОТА"
$env:TELEGRAM_OWNER_CHAT_ID="ТВОЙ_CHAT_ID"
npm start
```

Как узнать `TELEGRAM_OWNER_CHAT_ID`:

1. Напиши что-нибудь своему боту в Telegram.
2. Открой в браузере:

```text
https://api.telegram.org/bot<ТВОЙ_ТОКЕН_БОТА>/getUpdates
```

3. Найди в ответе поле `chat` -> `id`.
4. Это число и есть `TELEGRAM_OWNER_CHAT_ID`.
